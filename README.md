# DASHBOARD BIKE SHARRING

## Setup environment
```
conda create --name env_sub python=3.11
conda activate env_sub
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit
```

## Run steamlit app
```
streamlit run dashboard/dashboard.py
```
